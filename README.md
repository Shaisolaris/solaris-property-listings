# Solaris Estates — Property Listings

Modern property search for the Boston metro area. Real filters, real photo galleries, direct agent contact. The kind of search experience that doesn't make you bounce to Zillow.

**Live demo:** https://shaisolaris.github.io/solaris-property-listings/

## What it shows

- **12 properties** across Boston neighborhoods (Back Bay, Seaport, Beacon Hill, Cambridge, etc.)
- **Live filters** — search, type (House / Condo / Townhouse / Loft), bedrooms (range slider), max price (range slider)
- **Property detail modal** with gradient hero, specs grid, features, and direct agent contact form
- **Contact agent form** with success state
- **Dark mode** with localStorage persistence
- **Fully responsive** — mobile sheet-style modal, desktop grid
- **Featured / New badges**, per-property agent routing

## Stack

- Next.js 15 (App Router, static export)
- React 19 + TypeScript
- Tailwind CSS 3
- Deployed to GitHub Pages

## Run locally

```bash
npm install
npm run dev
```
